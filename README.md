# ✈️ Traveloop: India's Premium Travel Planner

Traveloop is a high-end, dark-themed travel planning prototype designed for the modern explorer. Built with a focus on premium aesthetics and seamless user experience, it allows users to plan, budget, and manage their journeys across India with ease.

---

## ✨ Key Features

- **Smart Trip Planning**: Generate detailed itineraries across 26+ Indian destinations.
- **Interactive Itinerary Builder**: Add/remove cities, manage stays, and pick from 20+ curated activities.
- **Budget Analytics**: Real-time spending tracker with visual charts and "Over Budget" alerts.
- **Premium Profile**: Track your travel stats, cities visited, and achievements in a sleek dashboard.
- **Checklist & Utilities**: Built-in packing checklists and travel journals to keep your trip organized.

---

## 🎨 Design System

Traveloop follows a strict **3-color brand identity** for a cohesive and premium look:

- 🌑 **Deep Black** (`#050508`) - For a focused, high-contrast dark environment.
- 💜 **Electric Violet** (`#6C47FF`) - Primary brand color for actions and highlights.
- 🌕 **Amber Gold** (`#FFC107`) - Secondary accent for status, luxury, and contrast.
